## AI 记忆系统研究

发现了几个重要的 GitHub 项目：

1. **mem0** - 通用记忆层
   - +26% 准确率 vs OpenAI Memory
   - 91% 更快响应，90% 更少 token
   - 多级记忆：用户、会话、Agent 状态

2. **memU** - 专为 24/7 主动 Agent 设计
   - 特别适合 OpenClaw 这类系统
   - 文件系统式记忆结构
   - 主动预测用户意图
   - 7×24 持续学习

3. **MemoRAG** - 基于记忆的 RAG
4. **Awesome-AI-Memory** - 资料汇总
